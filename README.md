# financial-data-analysis

## План проекта

Цель проекта — проанализировать акции российского рынка с помощью традиционных методов оценки финансовых инструментов и сравнить несколько подходов к выбору потенциально привлекательных бумаг.

1. **Выбрать финансовые данные**
   - использовать данные Московской биржи или финансовых агрегаторов;
   - рассмотреть акции первого и второго эшелона, то есть относительно ликвидные бумаги;
   - собрать рыночные данные: цену акции, объем торгов, динамику цены за выбранный период.

2. **Подготовить данные**
   - проверить данные на пропуски, дубликаты и некорректные значения;
   - привести даты, цены и финансовые показатели к удобному формату;
   - объединить рыночные данные с финансовыми индикаторами компаний.

3. **Провести первичный анализ рынка**
   - построить график изменения цены во времени;
   - рассчитать доходность акций;
   - оценить волатильность, то есть силу колебаний цены;
   - сравнить бумаги по динамике, риску и ликвидности.

4. **Рассчитать показатели фундаментального анализа**
   - оценить справедливую цену акции;
   - рассчитать число Грэма для поиска потенциально недооцененных бумаг;
   - использовать финансовые мультипликаторы, например P/E, P/B, P/S, ROE, долговую нагрузку и дивидендную доходность;
   - сравнить текущую рыночную цену с расчетными оценками.

5. **Сравнить стратегии отбора акций**
   - стратегия по справедливой цене: выбрать акции, у которых расчетная стоимость выше рыночной;
   - стратегия по числу Грэма: выбрать акции, которые выглядят недооцененными относительно этого показателя;
   - стратегия по мультипликаторам: выбрать компании с более привлекательными финансовыми коэффициентами;
   - сформировать итоговый рейтинг акций по выбранным критериям.

6. **Сделать выводы**
   - определить, какие акции выглядят наиболее привлекательными по результатам анализа;
   - сравнить, насколько отличаются результаты разных стратегий;
   - описать ограничения анализа, например зависимость от качества данных и выбранных показателей;
   - предложить возможные направления дальнейшего развития проекта.
