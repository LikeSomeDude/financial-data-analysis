# financial-data-analysis

## План проекта

Цель проекта — применить несколько простых методов прогнозирования к финансовым данным, сравнить качество полученных прогнозов и определить, какой подход дает наиболее точный результат.

1. **Выбрать финансовые данные**
   - определить источник данных;
   - выбрать финансовый показатель для прогноза, например цену закрытия акции, индекс или курс валюты;
   - описать период наблюдений и структуру датасета.

2. **Подготовить данные**
   - проверить данные на пропуски, дубликаты и некорректные значения;
   - привести даты и числовые показатели к удобному формату;
   - разделить данные на обучающую и тестовую части.

3. **Провести первичный анализ рынка**
   - построить график изменения цены во времени;
   - рассчитать доходность финансового инструмента;
   - оценить волатильность, то есть силу колебаний цены;
   - выделить заметные тренды, резкие падения или скачки.

4. **Построить несколько прогнозов**
   - наивный прогноз: следующее значение считается равным последнему известному значению;
   - прогноз с помощью скользящего среднего;
   - прогноз с помощью экспоненциального сглаживания;
   - прогноз с помощью линейной регрессии по времени.

5. **Сравнить качество методов**
   - построить графики фактических и прогнозных значений;
   - рассчитать ошибки прогноза с помощью метрик MAE, RMSE и MAPE;
   - сравнить результаты методов между собой.

6. **Сделать выводы**
   - определить, какой метод показал лучший результат на тестовых данных;
   - объяснить возможные причины различий между методами;
   - описать ограничения проекта и возможные направления дальнейшего улучшения.
